Privacy Policy

Last updated: June 18, 2026

At RU Player, we prioritize your privacy. This Privacy Policy describes how the application handles data when you use the software on your desktop computer (Windows) and mobile device (Android).

1. Local Playback & Zero Data Collection

RU Player operates completely on your local machine. We do not track, collect, or store any personal information, media playback history, file names, volume parameters, or visual filters. All playback details, playlists, and settings are saved locally on your device (in your local registry/AppData folders on Windows, or secure local storage on Android) and are never uploaded to our servers.

2. Platform System Permissions

The application requests system-level integrations which are handled strictly locally:

  • Windows App: Accesses the registry to set default file type associations (so that double-clicking video/audio files opens them in RU Player) and writes desktop/Start Menu shortcut files.
  • Android App: Requests storage read permissions (READ_MEDIA_VIDEO, READ_MEDIA_AUDIO, or READ_EXTERNAL_STORAGE) to scan local media files and present them in the media gallery. If permissions are denied, it uses the System Document Picker to let you choose files manually. No local files or media identifiers are transmitted.

3. Network Connectivity

  • Windows Update Checker: On startup, the Windows app queries a secure API endpoint (npoint.io) to check for updates. No personal identifiers or device telemetry details are sent during this request. If an update is selected, the player downloads the setup installer from a direct URL (e.g. GitHub Releases).
  • Android App: The Android application runs 100% offline and does not contain any background trackers, analytics, or ads.

4. Third-Party Engines

We utilize industry-standard, secure media pipelines:

  • Windows App uses the LibVLCSharp engine.
  • Android App uses Google's modern Android Media3 ExoPlayer engine.
Both engines run entirely offline on your device to parse and play local media files without telemetry or analytics tracking.

5. Children's Privacy

Because our software does not collect, store, or transmit any data of any kind, it is fully compliant with children's privacy regulations. We do not gather or target information from children under the age of 13.

6. Contact Us

If you have any questions or concerns regarding this Privacy Policy, please feel free to reach out to us at srzapps@gmail.com or Support@myruhi.in.
