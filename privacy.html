Privacy Policy

Last updated: June 18, 2026

At RU Player, we prioritize your privacy. This Privacy Policy describes how the application handles data when you use the software on your desktop computer.

1. Local Playback & Zero Data Collection

RU Player operates completely on your local machine. We do not track, collect, or store any personal information, media playback history, file names, volume parameters, or visual filters. All playback details, playlists, and settings are saved locally on your device (in your local registry or AppData folders) and are never uploaded to our servers.

2. Network Connectivity (Update Checker)

The application connects to the internet solely for the purpose of checking for software updates:

  • **Update Probing**: On startup, the application sends a secure HTTP request to our designated API endpoint (npoint.io) to fetch the latest version information. No personal identifiers or device telemetry details are sent during this request.
  • **Download**: If you choose to install an update, the player downloads the setup installer `.exe` from the direct download URL provided in the version metadata (e.g. GitHub Releases).

3. Third-Party Libraries

We utilize the **LibVLCSharp** library to parse and play media. While the engine handles network streaming files if you explicitly request them (e.g. streaming a network stream URL), the library does not collect or transmit analytics data in the background.

4. Windows System Permissions

The application requests system-level integrations which are handled locally:

  • **Registry Access**: Necessary to set default file associations so that Windows knows to launch RU Player when double-clicking video/audio files.
  • **Shortcut Creation**: Writes a shortcut file (`.lnk`) to your Desktop and Start Menu for ease of access.

5. Children's Privacy

Because our software does not collect, store, or transmit any data of any kind, it is fully compliant with children's privacy regulations. We do not gather or target information from children under the age of 13.

6. Contact Us

If you have any questions or concerns regarding this Privacy Policy, please feel free to reach out to us at srzapps@gmail.com or Support@myruhi.in.
